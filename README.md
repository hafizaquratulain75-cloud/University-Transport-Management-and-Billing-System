# University Transport Management and Billing System

A console-based University Transport Management and Billing System developed in C++ using Object-Oriented Programming (OOP) concepts.

## Project Overview

This system automates university transport operations including student registration, vehicle management, route allocation, transport pass management, billing, and reporting.

## Features

* Student and Admin Login
* Vehicle Management (Bus/Van)
* Route Management
* Transport Registration
* Bill Generation
* Revenue Reports
* Route Usage Reports
* File Handling for Data Persistence

## OOP Concepts Implemented

* Classes and Objects
* Inheritance
* Polymorphism
* Abstraction
* Encapsulation
* Composition
* Aggregation
* Association
* Operator Overloading
* Dynamic Memory Management
* File Handling
* Templates
